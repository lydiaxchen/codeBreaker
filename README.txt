Step 1: install node.js (maybe v8.15.1 or newer)
    sudo apt-get update
    sudo apt-get install curl
    curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
    sudo apt-get install nodejs
    
Step 2: install express
    npm install express
    
Step 3: run nodejs server
    node code_breaker_server.js

Step 4: change the url to http://localhost:3000/post
	open code_breaker_client.js, go to Line 11

Step 5: open the html file
    code_breaker.html

When putting the balls to the hole, please use the colorball at the bottom,
Not the balls on the board.
